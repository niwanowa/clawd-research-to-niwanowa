## 説明

Hugoで構築されたブログサイトです。技術的なTipsやメモをまとめたサイトとして運用されています。

公開URL: https://hugo.niwanowa.tips/

## 主な機能

- Hugoによる静的サイト生成
- Markdownベースの記事管理
- GitHub Actionsによる自動デプロイ

## 技術スタック

### フロントエンド
- Hugo（静的サイトジェネレーター）

### インフラ・ツール
- GitHub Actions
- AWS CloudFront
- AWS S3

## プロジェクト構成

```
niwanowa.tips/
├── content/           # 記事のMarkdownファイル
├── layouts/           # テンプレートファイル
├── static/            # 静的ファイル
├── config.toml        # Hugo設定ファイル
└── README.md
```

## 使い方

### 前提条件

- Hugo

### アップデート

#### Windows

```bash
CGO_ENABLED=1 go install -tags extended github.com/gohugoio/hugo@latest
```

#### macOS

```bash
brew upgrade hugo
```

### ローカルサーバー起動

```bash
cd niwanowa.tips/
hugo server -D
```
