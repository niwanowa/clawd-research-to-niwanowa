## 使い方

### インストール

```bash
git clone https://github.com/niwanowa/hello-world-pages.git
cd hello-world-pages
```

### 実行

GitHub Pagesで公開する場合：
1. GitHubリポジトリの「Settings」→「Pages」を開く
2. 「Source」を「Deploy from a branch」に設定
3. 「Branch」を「main」に設定
4. 保存後、自動的にデプロイされます
