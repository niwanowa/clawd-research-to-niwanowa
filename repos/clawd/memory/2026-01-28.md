## Slack スレッド返信設定の調査

nwianowa から Slack での応答をスレッドで行う設定について質問があった。

### 調査結果
- `preferThread` という設定は存在しない
- 代わりに `channels.slack.replyToMode` を使用する

### replyToMode オプション
| モード | 動作 |
| --- | --- |
| `off` | デフォルト。メインチャンネルに返信。トリガーがスレッド内だった場合のみスレッドに返信 |
| `first` | 最初の返信はスレッドへ、以降はメインチャンネルへ |
| `all` | すべての返信をスレッドで行う |

### 設定例
```json
{
  "channels": {
    "slack": {
      "replyToMode": "all"
    }
  }
}
```

### チャットタイプ別の設定も可能
`channels.slack.replyToModeByChatType` で DM / グループ / チャンネル別に設定可能
