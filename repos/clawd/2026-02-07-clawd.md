## 説明

OpenClaw用のAIアシスタント（Clawdbot）の設定ファイル集です。AGENTS.md、SOUL.md、TOOLS.md、USER.md、RESEARCH-GUIDE.mdなどの設定ファイルを管理しています。

## 主な機能

- AIアシスタントの人格設定（IDENTITY.md、SOUL.md）
- エージェント動作ガイドライン（AGENTS.md）
- ユーザー情報管理（USER.md）
- 調査レポート作成ガイド（RESEARCH-GUIDE.md）
- ツール固有設定（TOOLS.md）
- ハートビート管理（HEARTBEAT.md）

## 技術スタック

### コア
- OpenClawフレームワーク
- Markdown設定ファイル

### ドキュメント
- Markdown形式の設定管理

## プロジェクト構成

```
clawd/
├── AGENTS.md          # エージェント動作ガイドライン
├── SOUL.md            # AIアシスタントの人格定義
├── TOOLS.md           # ツール固有設定メモ
├── USER.md            # ユーザー情報
├── IDENTITY.md        # クラウドボットのアイデンティティ
├── RESEARCH-GUIDE.md  # 調査レポート作成ガイドライン
├── HEARTBEAT.md       # 定期チェックリスト
├── canvas/            # キャンバス用設定
├── memory/            # メモリ保存ディレクトリ
├── reports/           # レポート出力ディレクトリ
└── templates/         # テンプレートディレクトリ
```

## 使い方

### 設定ファイルの編集

各設定ファイルはMarkdown形式で記述されています。必要に応じて以下のファイルを編集します：

```bash
cd /path/to/clawd
# 設定ファイルを編集
vim AGENTS.md
vim SOUL.md
vim TOOLS.md
```

### 定期チェックの実行

HEARTBEAT.mdに記載されたチェックリストに従って定期的なメンテナンスを実施します。
